# 📊 Customer Feedback Analysis — Excel Project

An end-to-end Excel project analyzing customer feedback across multiple products. Covers data cleaning, sentiment classification, rating analysis, and an interactive dashboard — built to demonstrate practical data analyst skills.

---

## 🎯 Project Objectives

- Clean and standardize raw customer feedback data
- Classify feedback by sentiment (Positive / Neutral / Negative)
- Analyze product ratings and identify performance trends
- Break down feedback volume and sentiment by product
- Build a summary dashboard for quick insights

---

## 🔍 What's Inside the Workbook

| Sheet | Description |
|---|---|
| Raw Data | Original customer feedback records |
| Cleaned Data | Standardized and validated dataset ready for analysis |
| Sentiment Analysis | Sentiment classification and distribution breakdown |
| Product Feedback | Feedback count and sentiment split by product |
| Rating Analysis | Rating distribution across products |
| Dashboard | Visual summary of all key metrics |

## 🛠️ Tools & Techniques Used

- **Microsoft Excel**
  - Data Cleaning (removing duplicates, handling blanks, standardizing text)
  - Sentiment classification using formula logic
  - Pivot Tables for aggregation and cross-analysis
  - Charts (bar, pie, column) for visual storytelling
  - Dashboard design with slicers and linked visuals

---

## 💡 Key Insights

- Identified sentiment distribution across products to highlight which items received the most negative feedback
- Analyzed rating patterns to surface underperforming products
- Monthly trend analysis reveals shifts in customer satisfaction over time
